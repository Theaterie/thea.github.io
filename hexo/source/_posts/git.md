#### git使用指南

#### part1 环境配置

git log
查看提交历史
git status
检查目前Git仓库状态(nothing to commit, working tree clean)表示为正常的没有残留进程
git commit
提交信息
git commit -a
一口气全交了
git log --oneline
显示一行，简单主题
git shortlog
按用户显示每次提交主题
使用i进入编辑模式，esc退出编辑模式
:wq保存修改
